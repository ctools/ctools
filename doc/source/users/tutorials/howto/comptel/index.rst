Analysing COMPTEL data
----------------------

  .. admonition:: What you will learn

     You will learn how to analyse data from the COMPTEL telescope about NASA's
     `CGRO <https://heasarc.gsfc.nasa.gov/docs/cgro/cgro.html>`_
     satellite with ctools.

You can use ctools to analyse data from the COMPTEL telescope aboard NASA's
`CGRO <https://heasarc.gsfc.nasa.gov/docs/cgro/cgro.html>`_ satellite.
In this tutorial you will learn how to do that with a simple analysis.
