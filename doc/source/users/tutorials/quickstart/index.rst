First steps with ctools
-----------------------

This section provides a hands-on introduction of ctools. By means of an
example you will learn how to simulate and analyse CTA event data. This
section is intended to be read by newcomers. The reading and executing of the
examples should take about an hour, so grab some coffee before you start.
